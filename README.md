# Men's & Kid's Fashion E-Commerce Website

A responsive e-commerce website for men's & Kid's fashion. Users can browse products, add items to the cart, and complete purchases .  

## Features
- Browse products by category (shirts,hoodies ,t-shirts etc )
- Search and filter products
- Add products to cart and manage items
- User login and signup
- Responsive design for desktop and mobile

## Tech Stack
- Frontend: React.js, HTML, CSS
- Backend: Node.js
- Database: Firebase
- Version Control: Git & GitHub
